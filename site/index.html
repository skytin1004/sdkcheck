sdkcheck

Test product docs the way AI agents use them: execute the workflow, capture command evidence, and classify whether the docs, product, or environment failed.

View Repository Get Started

Run the first dogfood scenario

sdkcheck currently ships a Co-op Translator recipe that clones the project, creates an isolated Python environment, runs the documented CLI path, and writes Markdown plus JSON reports.

cargo install --locked --git https://github.com/skytin1004/sdkcheck.git sdkcheck

sdkcheck run --recipe co-op-translator --backend docker --fake-openai

Evidence for CI

Each report includes scenario steps, docs observations, secret names, command stdout/stderr, timeout state, generated files, failure classification, and reproduction command.

Built for API-key workflows

Secrets are passed only by explicit environment variable name. Values are masked before command output is written to reports.

Open source core

sdkcheck starts Apache-2.0 with a Rust CLI, Docker-first execution, GitHub release binaries, and a narrow MVP recipe.
